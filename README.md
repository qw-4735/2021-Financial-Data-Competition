# 2021-Financial-Data-Competition
> KB 국민카드 지역별 음식점 세부업종 집계내역 데이터를 활용한 음식점 업종 및 개업 위치 선정

* 기간 : 2021.07 ~ 2021.08
* 주관 : 2021 금융데이터 경진대회
* 데이터 수집 : 금융데이터거래소, 서울열린데이터광장, 경기데이터드림
* 역할 : 데이터 수집 및 가공, 인구 데이터 결합 및 전처리, 분석 방향 설계, 시각화
* 사용한 모델 : XGBoost
* 시각화 : 선정된 법정동
